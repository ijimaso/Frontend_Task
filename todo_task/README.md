## 自分で追加した機能
* 未完了/完了タスクの個数をリアルタイムで表示する機能
→タスクが多くなったときに，わざわざ下まで自分でスクロールして，個数を数える手間を減らしたいと考えたから．
* 不要なタスクを削除する機能
→不要なタスクを削除したいときがあるだろうと考えたから．
